# Future Interns – Prompt Engineering Task 1

## Project Title

AI-Powered Website Copy Generation System for Local Businesses

## Business Chosen

**Tarun Academy – Coaching Institute**

## Objective

The objective of this project is to design a structured prompt engineering framework that generates professional, conversion-focused website copy for local businesses. The system creates homepage content, service descriptions, and call-to-action sections that are ready for real-world website use.

## Prompt Logic

### 1. Business Context

Defines the business type, target audience, and location.

### 2. Benefit-Driven Messaging

Focuses on customer outcomes and value rather than features alone.

### 3. Conversion Optimization

Uses persuasive language and strong calls-to-action to improve engagement.

### 4. Reusability

The same prompt framework can be adapted for salons, cafes, clinics, agencies, freelancers, and coaching institutes.

## Tools Used

* ChatGPT

## Key Features

✅ Clear, benefit-driven website copy

✅ Business-specific language

✅ Strong call-to-action sections

✅ Adaptable and reusable prompts

✅ Real website-ready content
